# LumiqAI - Business Credit Visibility for Banks

## About

LumiqAI helps banks stop SMB deposit leakage by shipping business credit visibility and eligibility inside bank apps.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

## Build

```sh
npm run build
```
